# Django Blog Web Application

## 📌 Description

This is a Django-based blog project created as part of my learning. It allows users to explore blog posts by category and navigate through different pages of the website.
I built this project to understand Django concepts like models, views, templates, and static files.

---

## 🚀 Features

* View blog posts by category
* Responsive user interface using HTML, CSS, and JavaScript
* Static pages (About, Contact, FAQ, Privacy Policy)
* Image handling using media files
* Organized Django project structure

---

## 🛠️ Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite

---

## 📂 Project Structure

* `blog/` → Main app (models, views, urls)
* `iblogs/` → Project configuration
* `template/` → HTML templates
* `static/` → CSS and JavaScript files
* `media/` → Images used in blog

---

## ⚙️ How to Run the Project

1. Clone the repository

2. Install dependencies:
   pip install -r requirements.txt

3. Apply migrations:
   python manage.py migrate

4. Run the server:
   python manage.py runserver

5. Open in browser:
   http://127.0.0.1:8000/

---

## 📸 Notes
